I'm Mircea, a software engineer working mostly on decentralized identity infrastructure.

I build open-source tools that help developers work with
[DIDs](https://www.w3.org/TR/did-core/),
[Verifiable Credentials](https://www.w3.org/TR/vc-data-model/),
and related standards.

---

## Projects I maintain

<div class="project-cards">

<a class="project-card" href="https://github.com/decentralized-identity/veramo">
  <h3>Veramo</h3>
  <p>JavaScript framework for Verifiable Data.</p>
  <p>Create and manage DIDs, issue and verify credentials. Powers Node, browsers, and mobile apps.</p>
</a>

<a class="project-card" href="https://github.com/decentralized-identity/did-resolver">
  <h3>did-resolver</h3>
  <p>Universal DID resolution interface for JS/TS.</p>
  <p>Pluggable architecture for method-specific resolvers.</p>
</a>

<a class="project-card" href="https://github.com/decentralized-identity/ethr-did-resolver">
  <h3>ethr-did-resolver</h3>
  <p>DID resolver for the <code>did:ethr</code> method.</p>
  <p>Resolves DID Documents from ERC-1056 registry on Ethereum.</p>
</a>

<a class="project-card" href="https://github.com/decentralized-identity/web-did-resolver">
  <h3>web-did-resolver</h3>
  <p>DID resolver for the <code>did:web</code> method.</p>
  <p>Resolves DID Documents from HTTPS domains via <code>/.well-known/did.json</code>.</p>
</a>

<a class="project-card" href="https://github.com/decentralized-identity/did-jwt">
  <h3>did-jwt</h3>
  <p>Create and verify JWTs signed by DIDs.</p>
</a>

<a class="project-card" href="https://github.com/decentralized-identity/did-jwt-vc">
  <h3>did-jwt-vc</h3>
  <p>Create and verify W3C Verifiable Credentials and Presentations in JWT format.</p>
  <p>Built on did-jwt.</p>
</a>

</div>
